## Поиск генов, кодирующих факторы патогенности
Для анализа данных аннотации был написан [python-скрипт](group_annotation_data.py), группирующий данные [.gff файла](../annotation/annot.gff) и сохраняющий пригодную для чтения [таблицу](gene_table.tsv), содержащую колонки Locus_tag,	Product (белок, кодируемый геномом),	Protein_ID (идентификацилнный код в формате RefSeq), Start (порядковый номер нуклеотида, с которого начинается ген), End (порядковый номер нуклотида, которым ген заканчивается). 
Далее полученная таблица была отсортирована в [таблицу](pathogenic_gene.tsv), содержащую только "патогенные гены" (по ключевым словам) с помощью команды:
``` bash
grep -i -E "neurotoxin|botulinum|bont|ntnh|hemagglutinin|toxin.*complex" gene_table.tsv > pathogenic_gene.tsv.
```
